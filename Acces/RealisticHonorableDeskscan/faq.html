üåê General Questions

What is Access Network?
Access Network is a high-performance blockchain designed for speed, security, and scalability. It supports decentralized applications (dApps), smart contracts, and digital assets with minimal transaction fees and near-instant confirmations.
What is AccessScan?
AccessScan is the official block explorer for Access Network. It allows you to search and explore transactions, blocks, addresses, and smart contracts on the network in real-time.
What is the Chain ID for Access Network?
The Chain ID for Access Network is 22888 (or 0x5968 in hexadecimal).

To add Access Network to MetaMask:

1. Open MetaMask and click on the network dropdown
2. Click "Add Network" or "Add a network manually"
3. Enter the following details:
  ‚Ä¢ Network Name: Access Network
  ‚Ä¢ RPC URL: Your network's RPC endpoint
  ‚Ä¢ Chain ID: 22888
  ‚Ä¢ Currency Symbol: ACCESS

4. Click "Save"
Is Access Network compatible with Trust Wallet?
Yes! Access Network is fully EVM-compatible, which means it works with Trust Wallet, MetaMask, and other Ethereum-compatible wallets. Simply add the network using the custom network option in your wallet.

üìÑ Transactions

Why is my transaction pending?
Transactions may appear pending due to:

‚Ä¢ Low gas price - Try increasing the gas price
‚Ä¢ Network congestion - Wait for the network to process
‚Ä¢ Nonce issues - Check if previous transactions are confirmed

Note: AccessScan is a block explorer and cannot expedite, cancel, or replace transactions. Please contact your wallet provider for transaction assistance.
How fast are transactions on Access Network?
Access Network features ~3 second block times, providing near-instant transaction confirmations. Most transactions are confirmed within 1-2 blocks.
What are the transaction fees on Access Network?
Access Network offers very low transaction fees. The exact fee depends on the gas price and gas used by your transaction. Check the Gas Tracker for current gas prices.

Does Access Network have an API?
Yes! AccessScan provides a comprehensive API for developers. You can access transaction data, block information, address balances, and more. Visit our API Documentation for details.
Is Access Network EVM compatible?
Yes! Access Network is fully EVM (Ethereum Virtual Machine) compatible. This means you can deploy Solidity smart contracts, use Web3.js/Ethers.js libraries, and port existing Ethereum dApps with minimal modifications.
